# Privacy

Last updated: 2026-06-22

This repository is a public distribution surface for the Meridian Codex. It does not run an application. It does not set cookies. It does not collect analytics. It does not store private user data.

## What This Repository Collects

Nothing by itself.

The files in this repository are static text files. Reading, cloning, downloading, or forking the repository does not send data to the Meridian Codex beyond whatever GitHub itself records as part of operating github.com.

## GitHub Activity

If you star, watch, fork, clone over HTTPS, open an issue, submit a pull request, comment, or otherwise interact with this repository on GitHub, GitHub may collect and process information about that activity.

That data is handled by GitHub, not by this repository. See GitHub's Privacy Statement for GitHub's own terms: [https://docs.github.com/site-policy/privacy-policies/github-privacy-statement](https://docs.github.com/site-policy/privacy-policies/github-privacy-statement)

## Public Contributions

Issues, pull requests, commit metadata, and comments are public. Do not include secrets, API keys, private correspondence, confidential documents, personal data about other people, or anything you are not willing to publish.

If you quote private correspondence or identify a person, make sure you have the right to do so. The Codex's commitment to honest inquiry does not turn someone else's private information into public material.

## Website

The operational home of the Codex is [meridiancodex.com](https://meridiancodex.com). The website is a separate surface from this repository and may have its own privacy terms: [https://meridiancodex.com/privacy](https://meridiancodex.com/privacy)
